# <a name="optional-diagnostic-data-for-office"></a>ข้อมูลการวินิจฉัยเพิ่มเติมสำหรับ Office

> [!IMPORTANT]
> ข้อมูลในบทความนี้ใช้กับเวอร์ชัน 1904 ขึ้นไปของซอฟต์แวร์ไคลเอ็นต์ Office ต่อไปนี้ที่ติดตั้งบนคอมพิวเตอร์ที่ใช้ Windows:
> - Office 365 ProPlus และ Office 365 Business
> - Office 365 Personal, Office 365 Home หรือ Office เวอร์ชันอื่นที่เป็นส่วนหนึ่งของการสมัครใช้งาน Office 365
> - Project และ Visio ที่มีแผนการสมัครใช้งาน เช่น แผน Project Online Professional หรือ Visio Online Plan 2

ข้อมูลการวินิจฉัยมีไว้เพื่อรักษาความปลอดภัย อัปเดต ตรวจหา วินิจฉัย และแก้ไขปัญหาใน Office รวมถึงการปรับปรุงผลิตภัณฑ์ด้วย ข้อมูลนี้จะไม่มีชื่อหรือที่อยู่อีเมลของผู้ใช้ เนื้อหาไฟล์ของผู้ใช้ หรือข้อมูลเกี่ยวกับแอปที่ไม่เกี่ยวข้องกับ Office

ข้อมูลการวินิจฉัยนี้จะรวบรวมและส่งถึง Microsoft เกี่ยวกับซอฟต์แวร์ไคลเอ็นต์ Office ที่ทำงานบนคอมพิวเตอร์ที่ใช้ Windows จำเป็นต้องระบุข้อมูลการวินิจฉัยบางอย่าง ขณะที่ข้อมูลการวินิจฉัยบางอย่างสามารถระบุหรือไม่ก็ได้ เราให้ความสามารถในการเลือกว่าจะส่งข้อมูลการวินิจฉัยที่จำเป็นหรือเพิ่มเติมให้เราผ่านการใช้การควบคุมความเป็นส่วนตัวหรือไม่ เช่น การตั้งค่านโยบายสำหรับองค์กร คุณสามารถดูข้อมูลการวินิจฉัยที่ส่งถึงเราโดยใช้ตัวแสดงข้อมูลการวินิจฉัย

***ข้อมูลการวินิจฉัยเพิ่มเติม*** เป็นข้อมูลเพิ่มเติมที่ช่วยให้พวกเราสามารถปรับปรุงผลิตภัณฑ์และให้ข้อมูลที่ผ่านการเพิ่มประสิทธิภาพเพื่อช่วยให้เราตรวจหา วินิจฉัย และแก้ไขปัญหาได้

ถ้าคุณเลือกส่งข้อมูลการวินิจฉัยเพิ่มเติมถึงเรา จะมีข้อมูลการวินิจฉัยที่จำเป็นรวมอยู่ด้วย

ตัวอย่างของข้อมูลการวินิจฉัยเพิ่มเติมมีข้อมูลที่เรารวบรวมเกี่ยวกับรูปภาพที่ผู้ใช้แทรกลงในเอกสาร Word เพื่อให้เราสามารถมอบตัวเลือกรูปที่ดีขึ้น และข้อมูลที่เรารวบรวมเกี่ยวกับเวลาที่ใช้ในการแสดงสไลด์ PowerPoint บนหน้าจอของคุณเพื่อให้เราสามารถปรับปรุงประสบการณ์ใช้งานได้ถ้าทำงานช้า

สำหรับข้อมูลเพิ่มเติมเกี่ยวกับข้อมูลการวินิจฉัย ให้ดูที่หัวข้อต่อไปนี้:

- [ข้อมูลการวินิจฉัยที่จำเป็นสำหรับ Office](required-diagnostic-data.md)
- [การใช้ตัวแสดงข้อมูลการวินิจฉัยกับ Office](https://support.office.com/article/cf761ce9-d805-4c60-a339-4e07f3182855)

ถ้าคุณเป็นผู้ดูแลระบบสำหรับองค์กรของคุณ คุณอาจสนใจหัวข้อต่อไปนี้:

- [ภาพรวมของการควบคุมความเป็นส่วนตัวของ Office 365 ProPlus](overview-privacy-controls.md)
- [ใช้การตั้งค่านโยบายเพื่อจัดการการควบคุมความเป็นส่วนตัวของ Office 365 ProPlus](manage-privacy-controls.md)

## <a name="categories-of-optional-diagnostic-data"></a>ประเภทของข้อมูลการวินิจฉัยเพิ่มเติม

ข้อมูลการวินิจฉัยเพิ่มเติมจัดแบ่งเป็นประเภทต่อไปนี้:

- การตั้งค่าซอฟต์แวร์และสินค้าคงคลัง
- การใช้งานบริการและผลิตภัณฑ์
- ประสิทธิภาพของบริการและผลิตภัณฑ์
- การเชื่อมต่อและการกำหนดค่าของอุปกรณ์

ประเภทเหล่านี้จะแสดงอยู่ในตัวแสดงข้อมูลการวินิจฉัย และเป็นประเภทเดียวกันกับที่ใช้กับข้อมูลการวินิจฉัยที่จำเป็น

ส่วนต่อไปนี้มีคำอธิบายของแต่ละประเภทและตัวอย่างของเหตุการณ์สำหรับแต่ละประเภท

## <a name="software-setup-and-inventory-events"></a>เหตุการณ์ของการตั้งค่าซอฟต์แวร์และสินค้าคงคลัง

ประเภทนี้มีเหตุการณ์ที่อาจครอบคลุมพื้นที่ต่อไปนี้:

- ผลิตภัณฑ์และเวอร์ชันที่ติดตั้งและสถานะการติดตั้ง
- Add-in ของซอฟต์แวร์และการตั้งค่า
- เงื่อนไขของข้อผิดพลาดสำหรับเอกสาร ฟีเจอร์ และ Add-in ที่อาจลดการรักษาความปลอดภัยลง รวมถึงความพร้อมในการอัปเดตผลิตภัณฑ์

ตารางต่อไปนี้แสดงตัวอย่างของเหตุการณ์ในประเภทนี้และคำอธิบายของเหตุการณ์เหล่านั้น

| **ชื่อเหตุการณ์**   | **คำอธิบายเหตุการณ์**  |
| ---- | ---- |
| Office\_Extensibility\_AppCommands\_GetRibbonUpdatesForUserId | เหตุการณ์นี้ระบุว่า Word อัปเดต Ribbon ในส่วนติดต่อผู้ใช้ของ Word ได้สำเร็จหรือไม่เมื่อผู้ใช้เปลี่ยนแปลงข้อมูลประจำตัวของตน เราใช้เหตุการณ์นี้เพื่อตรวจสอบการตั้งค่าที่ไม่ถูกต้องและปัญหาอื่นๆ ที่อาจส่งผลต่อส่วนติดต่อผู้ใช้ของ Office |
| Office.Extensibility.AppCommands.AppCmdInstall   | เหตุการณ์นี้มีข้อมูลเกี่ยวกับ Add-in ของ Office ที่ผู้ใช้ติดตั้ง รวมถึง ID แอป รุ่นและเวอร์ชันของระบบปฏิบัติการ ความสำเร็จของการติดตั้ง และระยะเวลาของการติดตั้ง  |

## <a name="product-and-service-usage-events"></a>เหตุการณ์ของการใช้งานบริการและผลิตภัณฑ์

ประเภทนี้มีเหตุการณ์ที่อาจครอบคลุมพื้นที่ต่อไปนี้:

- ความสำเร็จของฟังก์ชันการทำงานของแอปพลิเคชัน จำกัดเฉพาะการเปิดและปิดแอปพลิเคชันและเอกสาร การแก้ไขไฟล์ และการแชร์ไฟล์ (การทำงานร่วมกัน)
- การกำหนดว่าเหตุการณ์ของฟีเจอร์เฉพาะเกิดขึ้นหรือไม่ เช่น เริ่มหรือหยุด และฟีเจอร์กำลังทำงานอยู่หรือไม่
- ฟีเจอร์การช่วยสำหรับการเข้าถึง Office

ตารางต่อไปนี้แสดงตัวอย่างของเหตุการณ์ในประเภทนี้และคำอธิบายของเหตุการณ์เหล่านั้น

| **ชื่อเหตุการณ์**   | **คำอธิบายเหตุการณ์**  |
| ------ | ------- |
| Office.Word.Commanding.Highlight  | เหตุการณ์นี้ระบุว่า Word ได้ดำเนินการคำสั่งเพื่อเน้นข้อความ เราใช้เหตุการณ์นี้เพื่อตรวจสอบข้อผิดพลาดในคำสั่งการเน้นข้อความ  |
| Office.Translator.AddInLoaded   | ฮาร์ทบีทเพื่อระบุว่าฟีเจอร์ตัวแปลภาษาถูกโหลด และแสดงผลสำเร็จแล้ว  |
| Office.Graphics.InsertPictureCommandActivity  | ติดตามความสำเร็จหรือความล้มเหลวของฟีเจอร์แทรกรูปภาพ และยังรายงานรายละเอียดเกี่ยวกับชนิดของรูปภาพที่แทรกและแหล่งที่มา|
| Office.PowerPoint.PPT.Desktop.SummaryZoomInsertionRule   | เหตุการณ์นี้กำหนดว่ามีส่วนใดส่วนหนึ่งแสดงอยู่ในเอกสารเมื่อผู้ใช้แทรกการซูมสรุปหรือไม่ และผู้ใช้เลือกที่จะลบส่วนที่มีอยู่หรือไม่ |
| Office.Security.SecureReaderHost.ProtectedViewValidation | ติดตามเวลาและสาเหตุที่มีการเปิดไฟล์ในมุมมองที่ได้รับการป้องกัน ใช้กับเงื่อนไขการวินิจฉัยที่มุมมองที่ได้รับการป้องกันอาจทริกเกอร์ไม่ถูกต้องเพื่อให้แน่ใจว่าฟีเจอร์ทำงานอย่างถูกต้อง |

## <a name="product-and-service-performance-events"></a>เหตุการณ์ของประสิทธิภาพของบริการและผลิตภัณฑ์

ประเภทนี้มีเหตุการณ์ที่อาจครอบคลุมพื้นที่ต่อไปนี้:

- แอปพลิเคชันจบการทำงานโดยไม่คาดคิด (การหยุดทำงาน) และสถานะของแอปพลิเคชันเมื่อเกิดเหตุการณ์
- เวลาตอบสนองหรือประสิทธิภาพการทำงานต่ำสำหรับสถานการณ์ เช่น การเริ่มต้นแอปพลิเคชัน หรือการเปิดไฟล์
- ข้อผิดพลาดในฟังก์ชันการทำงานของฟีเจอร์หรือประสบการณ์ใช้งานของผู้ใช้

ตารางต่อไปนี้แสดงตัวอย่างของเหตุการณ์ในประเภทนี้และคำอธิบายของเหตุการณ์เหล่านั้น

| **ชื่อเหตุการณ์**    | **คำอธิบายเหตุการณ์**   |
| --------------- | -------------- |
| Office\_Word\_Word\_CoreSaveTime100ns     | เหตุการณ์นี้บันทึกประสิทธิภาพการทำงานของกิจกรรมการบันทึกเอกสารโดย Word เราใช้เหตุการณ์นี้เพื่อตรวจสอบข้อผิดพลาดและปัญหาประสิทธิภาพการทำงานในกิจกรรมการบันทึกเอกสารของ Word|
| Office.Identity.SignInForWamAccountAad  | เหตุการณ์นี้จะถูกส่งเมื่อผู้ใช้ลงชื่อเข้าใช้บัญชี Azure Active Directory ที่มีไลบรารีระบบจัดการบัญชีบนเว็บ (WAM) เหตุการณ์นี้จะส่งเมตาดาต้า เช่น AppName, AppVersion และ ErrorCode ถ้าเหตุการณ์ล้มเหลว |
| Office.PowerPoint.PPT.Desktop.FileOpen.FirstSlideMasterThumbnailRenderTime | เหตุการณ์นี้รวบรวมระยะเวลาที่ใช้ในการแสดงรูปขนาดย่อต้นแบบของสไลด์แรกใน PowerPoint  |
| Office.Extensibility.Diagnostics   | เหตุการณ์นี้มีข้อมูลการวินิจฉัยทั่วไปสำหรับ Add-in ของ Office เช่น รายงานการหยุดทำงานสำหรับการแก้จุดบกพร่อง|

## <a name="device-connectivity-and-configuration-events"></a>เหตุการณ์ของการเชื่อมต่อและการกำหนดค่าของอุปกรณ์

ประเภทนี้มีเหตุการณ์ที่อาจครอบคลุมพื้นที่ต่อไปนี้:

- สถานการเชื่อมต่อเครือข่ายและการตั้งค่าอุปกรณ์ เช่น หน่วยความจำ

ตารางต่อไปนี้แสดงตัวอย่างของเหตุการณ์ในประเภทนี้และคำอธิบายของเหตุการณ์เหล่านั้น

| **ชื่อเหตุการณ์**                    | **คำอธิบายเหตุการณ์**                                                                                                                                                     |
| ------ | ----- |
| Office\_Graphics\_ArtViewValidate | เหตุการณ์นี้บันทึกการตรวจสอบผลลัพธ์ของมุมมองกราฟิกที่สนับสนุนส่วนติดต่อผู้ใช้ของกราฟิก เราใช้เหตุการณ์เพื่อรวบรวมข้อมูลการใช้งานและข้อผิดพลาดเกี่ยวกับการแสดงผลกราฟิก |
| Office.Graphics.ARCExceptionScope | เหตุการณ์นี้ติดตามความล้มเหลวในการแสดงผลที่มาจากโปรแกรมการแสดงผล |
| Office.Extensibility.ODPLatency   | เหตุการณ์นี้มีข้อมูลเกี่ยวกับการเชื่อมต่อเครือข่ายและความเร็วของผู้ใช้     |
